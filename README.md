# 📝 Task Manager Fullstack

Aplicação fullstack para gerenciamento de tarefas.

## 🚀 Tecnologias

* Node.js
* TypeScript
* Express
* Vue 3
* Axios

## 📂 Estrutura do projeto

* `/backend` → API REST
* `/frontend` → Interface web

## ⚙️ Como rodar

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🌐 Funcionalidades

* Criar tarefas
* Listar tarefas
* Remover tarefas

## 📡 Endpoints

* GET /tasks
* POST /tasks
* DELETE /tasks/:id
